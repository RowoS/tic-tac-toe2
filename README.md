# React + TypeScript + Vite Tic-Tac-Toe

This TicTacToe game was made using React and Vite + Typescript, with the aid of the [Tutorial Guide on Making Tic-Tac-Toe in React](https://react.dev/learn/tutorial-tic-tac-toe).

## Features
- Play a simple game of Tic-Tac-Toe
- Undo moves or reset your game
